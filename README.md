# Inhabitent Camping Supply Co.

A camping supply website, which includes a Blog, a Shop section, an About page, a Find Us page, and a serach function.

![](Giphy1.gif)

![](Giphy2.gif)

## Technologies Used

- HTML
- CSS
- JavaScript
- JQuery
- PHP
- Gulp
- Wordpress

## Installation

**Run locally with Gulp:**

```bash
npm install
```

**Build for production:**

```bash
npx gulp
```
